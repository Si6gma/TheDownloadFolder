# TechBlog Template

A responsive blog website template built with Bootstrap.

## What It Is

A multi-page blog template featuring various post formats (standard, gallery, audio, video) with a clean, modern design.

## Tech Stack

- HTML5
- CSS3 (with custom styles)
- JavaScript
- Bootstrap framework
- Font Awesome icons

## Project Structure

```
TheDownloadFolder/
├── index.html              # Homepage
├── about.html              # About page
├── contact.html            # Contact page
├── single-*.html           # Post format templates
├── css/                    # Stylesheets
├── js/                     # JavaScript files
├── fonts/                  # Icon fonts
├── images/                 # Image assets
└── media/                  # Media assets
```

## How to Use

1. Open `index.html` in a web browser
2. Customize content in HTML files
3. Replace images in the `images/` folder
4. Modify styles in `css/` folder as needed

## Key Learnings

- Bootstrap grid system for responsive layouts
- Multi-page website structure and navigation
- Template-based content organization
